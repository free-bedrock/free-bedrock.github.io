Openplace
The Java experience, for Bedrock.

Openplace is a free and adless addon marketplace for Minecraft Bedrock Edition. Every addon here is built to bring quality-of-life, competitive, and casual features from Java Edition — without paywalls, coins, or ads.

The project is open source and welcomes contributors of all skill levels. If you build addons, fix bugs, or write docs, you belong here.
