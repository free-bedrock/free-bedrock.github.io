Bedrock Lockout

Bedrock Lockout is a high-stakes competitive mode inspired by SmallAnt's popular Java mod. Two or more players (and in case you don't have friends, there is a solo mode as well) race to claim tasks on a shared Bingo board; once a task is claimed, the opponent can never get it.
